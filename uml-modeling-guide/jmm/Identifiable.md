# Stereotype `«Identifiable»`
Stereotype «Identifiable» can be used to mark a class to have an ID.

<br>

| **Stereotype**          | `«Identifiable»` | |
| ----------------------- | -------------- | |
| **Applicable Elements** | `Class`        |
| **Tagged Values**       | none           |
